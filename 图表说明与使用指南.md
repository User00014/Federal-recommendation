# 图表说明与使用指南（中文增强版）

## 1. 数据读取
- 当前脚本只读取 `logs/` 目录下的实验日志。
- 其他目录（如 `最终数据`、`备用数据`）不会参与绘图。
- 历史探索性图表已归档到 `figures/archive_exploratory_20260309/`。

## 2. 图例与标注策略
- 主要对比图均采用“主图 + 独立图例子图”的布局，避免图例与曲线重叠。
- 日志选择器会显示每个实验的颜色 `color=#xxxxxx`，便于论文中对照说明。
- 同模型不同参数也会分配不同颜色，提高可读性。

## 3. 核心图（含训练时间）
- `archive_exploratory_20260309/fig1_core_curves.png`：RMSE/ASR/AUC 三联曲线。
- `archive_exploratory_20260309/fig2_pareto_bubble.png`：隐私-效用 Pareto 气泡图（气泡大小对应平均每轮耗时）。
- `archive_exploratory_20260309/fig3_tail_asr_boxstrip.png`：尾段 ASR 分布（箱线 + 散点）。
- `archive_exploratory_20260309/fig4_noise_schedule.png`：有效噪声强度 `sigma*` 调度曲线。
- `archive_exploratory_20260309/fig5_round_time_curve.png`：每轮训练时间曲线（Round Time，已恢复）。

## 4. 统计图
- `archive_exploratory_20260309/fig6_score_heatmap.png`：创新综合评分热力图。
- `archive_exploratory_20260309/fig7_metric_correlation.png`：指标相关性热力图。
- `archive_exploratory_20260309/fig8_grouped_statistics.png`：按 `mode-algo-adp` 分组统计图。
- `archive_exploratory_20260309/fig9_tail_ci_forest.png`：尾段均值 ±95%CI 森林图。

## 5. 菜单操作建议
1. 先用 `s` 选择参与绘图的日志（或 `a` 全选）。
2. 用 `1` 先生成核心五图（包含训练时间）。
3. 再按 `7/8/9/0` 查看统计层面的结论。
4. 用 `e` 导出当前选择的汇总 CSV。
